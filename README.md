# VoxelGL

This Project converts .obj files to voxel scenes and renders them using OpenGL.

![alt text](https://github.com/xPasquale1/VoxelGL/blob/main/images/ref.png "Sponza")

# Features

Font rendering using .ttf
Decent memeory usage
Direct sun Lighting
Probe based global illumination
Raytraced single (or double) bounce GI

# Installation

Written with C++ for Windows. Tested only with the GCC compiler.
```
-lopengl32
-lgdi32
```
